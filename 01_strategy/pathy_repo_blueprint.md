# Pathy — Blueprint du dépôt privé

Date: 2026-05-28

## 1) Rôle du dépôt

Le dépôt privé Pathy est le point central de pilotage personnel.
Il agrège :
- la mémoire,
- les priorités,
- les décisions,
- les projets,
- les relations,
- les synthèses,
- les synchronisations avec Opays et Commons.

## 3) Ce que chaque dossier contient

### 00_governance
Les règles du système :
- ce qu’on garde,
- ce qu’on refuse,
- ce qui est prioritaire,
- comment on révise.

### 01_memory
La mémoire utile :
- personnes,
- projets,
- décisions,
- partenaires,
- historique.

### 02_dashboards
Les vues de pilotage :
- quotidien,
- hebdomadaire,
- mensuel,
- priorités.

### 03_agents
Les rôles des agents Hermes dans Pathy.

### 04_projects
La vision de tes projets majeurs.

### 05_sync
Ce qui est synchronisé avec :
- Opays Business,
- Opays Commons,
- contextes externes.

### 06_templates
Les formats standard pour ne pas réinventer les documents.

---

## 4) Règles de synchronisation

### Opays Business
Pathy reçoit :
- synthèses,
- décisions,
- alertes,
- priorités,
- résumé de réunions.

### Opays Commons
Pathy reçoit :
- calendrier d’événements,
- partenaires,
- contribution ouverte,
- besoins de financement,
- métriques d’impact.

### Vie personnelle
Pathy reçoit :
- obligations personnelles,
- tâches,
- relations,
- engagements.

---

## 5) Ce qui doit être automatisé

- synthèse quotidienne,
- synthèse hebdomadaire,
- registre des décisions,
- suivi des relances,
- détection des doublons,
- classement des tâches.

---

## 6) Ce qui doit rester manuel

- validation des décisions importantes,
- engagement financier,
- arbitrage stratégique,
- changements de direction,
- relations sensibles.

---

## 7) Règle de départ

Le dépôt doit rester :
- simple,
- lisible,
- privé,
- utile,
- et facile à maintenir.
