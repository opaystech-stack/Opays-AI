# Opays Tech — Hermes Agent

Quand tu travailles sur des interfaces utilisateur, du design UX/UI, ou des composants frontend, charge le skill `product-design-ux-ui` :

```
/skill product-design-ux-ui
```

Ce skill est basé sur la méthodologie Vercel (agent skills + lint rules + evals + human review loop). Il contient des règles pour :
- Formulaires (labels, erreurs, placeholders)
- Navigation (aria-current, menu mobile)
- Modales (focus trap, fermeture)
- Copie & microcopie (ton Opays Tech, CTA)
- Responsive & mobile (mobile-first, tap targets)
- Accessibilité (WCAG 2.2)
- États (loading, empty, error, disabled)

Voir `AGENTS.md` pour les autres règles de travail.
