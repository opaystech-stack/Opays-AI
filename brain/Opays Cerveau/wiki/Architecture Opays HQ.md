# Architecture Opays HQ

## Résumé
Opays HQ est la plateforme interne de gestion ("Command Center") développée pour piloter les opérations d'OPAYS TECH.

## Stack Technique
- **Framework** : Next.js 16 (App Router)
- **UI/UX** : React 19, Tailwind CSS, Framer Motion (Animations premium), Lucide Icons.
- **Backend & Auth** : Supabase (PostgreSQL, SSR Auth).
- **IA** : Vercel AI SDK (@ai-sdk/openai, @ai-sdk/react).
- **Validation** : Zod.

## Composants Clés
- **Espace Audit IA** : Interface pour monitorer et piloter les agents.
- **Gestion d'Equity** : Système de distribution et de suivi des parts pour les associés.
- **Modals** : Système de fenêtres modales pour les opérations rapides (Partners, Contracts, Tasks).
