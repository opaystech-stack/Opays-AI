# Skill : Plan Progress

## Role
Compare l'état réel du projet avec le plan 90 jours et les décisions validées.

## Entrées
- `06_ops/plan_execution_refonte_90_jours.md`
- `05_decisions/decision_log.md`
- état Git / issues / services déployés / checklists internes

## Fonction
- calculer l'avancement par phase,
- signaler les écarts,
- identifier les tâches bloquées,
- repérer les dépendances non traitées,
- estimer ce qui peut être livré maintenant.

## Sortie attendue
- progression par phase,
- éléments terminés,
- éléments en retard,
- éléments à risque,
- prochaine action prioritaire.
