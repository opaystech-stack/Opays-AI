# Skill : Web Artifacts Builder (Anthropics Standard)

## Rôle
Créer des artefacts HTML/React complexes, multi-composants, utilisant les technologies web modernes (React, Tailwind CSS, shadcn/ui).

## Principes
1. **Complexité Maîtrisée :** Utiliser pour des dashboards, des calculateurs interactifs ou des visualisations de données nécessitant une gestion d'état.
2. **Stack :** React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui.
3. **Single File :** Capacité à regrouper tout le code (JS, CSS, dépendances) dans un seul fichier HTML auto-contenu.

## Guidelines Design
- Éviter le "AI slop" : layouts trop centrés, gradients violets systématiques, coins arrondis uniformes partout.
- Utiliser le système de thémage de shadcn/ui pour une cohérence totale.

## Workflow
- Initialiser le projet avec une structure propre (aliases `@/`).
- Développer de manière modulaire.
- Packager pour un affichage fluide en tant qu'artefact.
