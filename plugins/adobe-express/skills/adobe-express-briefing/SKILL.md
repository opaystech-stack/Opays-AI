# Adobe Express Briefing

Ce skill aide à transformer une intention brute en contenu exploitable dans Adobe Express.

## Objectif

- clarifier le message,
- structurer le contenu,
- préparer des variantes simples,
- éviter les prompts vagues ou trop larges.

## Quand l’utiliser

- pour une publication sociale,
- pour une bannière,
- pour une landing visuelle,
- pour une affiche ou une création simple,
- pour reformuler un message en version courte et crédible.

## Méthode

1. Identifier le support.
2. Définir le but.
3. Déterminer le public.
4. Fixer le message principal.
5. Proposer une structure visuelle et textuelle.
6. Lister les variantes courtes.

## Format de sortie recommandé

```text
Support:
Objectif:
Audience:
Message principal:
Accroche:
Sous-texte:
CTA:
Variantes courtes:
Notes visuelles:
```

## Règles

- rester simple,
- éviter le jargon,
- privilégier les messages courts,
- garder une hiérarchie claire,
- toujours relier le visuel à l’intention.
