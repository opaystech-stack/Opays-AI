# OPAYS TECH — CHARTE GRAPHIQUE & VISUELLE
### Guide de référence pour toute création visuelle (images, flyers, brochures, vidéos, UI)

---

## 1. PALETTE DE COULEURS

| Nom                | Code HEX  | Usage                                    |
|--------------------|-----------|------------------------------------------|
| Noir Profond       | `#0A0A0A` | Fond principal (Dark Mode obligatoire)   |
| Gris Anthracite    | `#1C1C1C` | Fond secondaire, cartes, surfaces        |
| Cyan Électrique    | `#00FFE0` | Accent principal, lignes de données, CTA |
| Bleu Cobalt        | `#0055FF` | Accent secondaire, boutons, gradients    |
| Blanc Pur          | `#FFFFFF` | Texte principal sur fond sombre          |
| Gris Clair         | `#A0A0A0` | Texte secondaire, sous-titres            |

### Règles :
- Le fond est **toujours** sombre. Jamais de fond blanc ou clair.
- Les accents Cyan et Bleu Cobalt ne doivent jamais être saturés au point de fatiguer l'œil. Rester subtil.
- Les gradients vont du Noir vers le Bleu Cobalt ou du Gris Anthracite vers le Cyan (jamais l'inverse).

---

## 2. TYPOGRAPHIE

| Usage          | Police       | Poids         |
|----------------|-------------|---------------|
| Titres         | **Inter**    | Bold (700)    |
| Sous-titres    | **Inter**    | SemiBold (600)|
| Corps de texte | **Inter**    | Regular (400) |
| Code / Data    | **JetBrains Mono** | Regular (400) |

### Règles :
- Taille minimum lisible : 14px pour le corps, 28px pour les titres sur les visuels.
- Espacement des lettres (tracking) : serré pour les titres, normal pour le corps.

---

## 3. STYLE VISUEL

### Esthétique générale :
- **Dark Mode Premium :** Toujours. Pas de mode clair.
- **Glassmorphism :** Surfaces en verre givré semi-transparent avec un léger flou (backdrop-blur).
- **Minimalisme :** Pas de surcharge. Chaque élément doit avoir une raison d'exister.
- **Éclairage Cinématographique :** Lumières directionnelles (top-down ou latérales), ombres douces, reflets subtils.
- **Précision Chirurgicale :** Les lignes sont nettes, les angles sont définis, les données sont propres.

### Textures autorisées :
- Verre givré (glassmorphism)
- Métal brossé
- Néon subtil (lignes fines, pas de glow excessif)
- Circuits imprimés stylisés (très discrets)

---

## 4. INTERDICTIONS VISUELLES

> Ces éléments sont **formellement interdits** dans toute création Opays Tech :

- ❌ Robots humanoïdes ou androïdes
- ❌ Cerveaux bleus lumineux (cliché IA)
- ❌ Mains de robot serrant des mains humaines
- ❌ Globe terrestre avec des connexions (cliché réseau)
- ❌ Fond blanc ou clair
- ❌ Couleurs vives non contrôlées (rouge vif, vert fluo, jaune)
- ❌ Éléments "Matrix" (cascade de caractères verts)
- ❌ Stock photos génériques de bureaux

---

## 5. LOGO & IDENTITÉ

### Logo Opays Tech :
- Fichier de référence : `png logo opays.png`
- Le logo doit toujours être affiché sur fond sombre.
- Espace de respiration minimum : la hauteur du logo de chaque côté.
- Ne jamais déformer, incliner, ou recolorer le logo.

### Sceau administratif :
- Fichier : `sceau admin opays.png`
- Usage réservé aux documents officiels (contrats, lettres, fiches de poste).

---

## 6. MOTS-CLÉS VISUELS (pour les prompts d'images)

Utiliser systématiquement ces termes dans les prompts de génération d'images :

```
dark mode, premium, cinematic lighting, glassmorphism, cyan accent,
cobalt blue, minimalist, hyper-detailed, corporate tech aesthetic,
data streams, holographic UI, sovereign infrastructure, clean lines,
negative space, 8k resolution
```

### Mots-clés à éviter :
```
robot, brain, matrix, colorful, cartoon, flat design, playful,
bright background, stock photo, generic
```
