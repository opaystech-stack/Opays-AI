## PROMPT 1 : GÉNÉRATION D'IMAGES MARKETING (Créatif)

```
Rôle : Tu es un Directeur de Création de classe mondiale spécialisé dans le 
lancement de "Deep Tech" souveraines.

Ta Mission : En te basant sur l'identité profonde d'Opays Tech (consulte le 
Livre Blanc, la structure Studio/Labs, et le concept d'Ingénierie de 
l'Efficience par l'IA), conçois des visuels marketing (flyers de lancement, 
dépliant stratégique, image de marque).

Contraintes de Créativité :
1. Interprétation Libre : Ne crée pas de visuels génériques. Propose une 
   esthétique qui incarne visuellement la "suppression de la friction" et la 
   "souveraineté technologique".
2. Ancrage Projet : Tes créations doivent refléter la dualité entre l'impact 
   ROI immédiat (Studio) et la recherche d'intérêt général (Labs).
3. Branding Haut de Gamme : Respecte l'esthétique premium du projet (Dark mode, 
   accents cyan/électriques, minimalisme, précision chirurgicale).
4. Échelle d'Intervention : Le visuel doit suggérer que l'entreprise intervient 
   aussi bien sur des audits d'entreprises privées que sur des infrastructures 
   à caractère national.

Surprends-moi par ta vision artistique de l'Efficience.
```

## PROMPT 2 : ANIMATION PUBLICITAIRE (HypeFrames / HeyGen)

```
Create a 15-second premium advertising animation for Opays Tech (deep-tech, 
AI-driven efficiency, national sovereignty).

Branding & style:
- Dark-mode background (deep black #0A0A0A) with subtle animated gradient.
- Accent colours: cyan #00FFE0 and electric blue #0055FF.
- Minimalist, cinematic, photorealistic textures (glass-morphed server, 
  metal-brushed surfaces).
- Clean, modern typography: Inter (white, 28pt) for headlines, Outfit 
  (light gray) for sub-text.

Visual storyline (scene-by-scene):
1. 0-3s – A futuristic glass-like server pulses cyan light; data-streams flow 
   outward.
2. 3-6s – Split-screen: left = Studio dashboard with KPI graphs (ROI); 
   right = Labs with glowing data-center and stylised DRC map. A thin cyan 
   filament links both halves.
3. 6-9s – Camera pans to a metal-brushed hand touching a holographic panel; 
   chaotic red chart turns into smooth cyan line.
4. 9-12s – Text overlay (fade-in): "Transformez la complexité en fluidité." 
   "ROI mesurable – souveraineté assurée."
5. 12-15s – Company logo appears central, surrounded by animated neon vortex; 
   tagline: "L'IA au service de l'efficacité nationale."

Audio & timing:
- Ambient synth-pad with subtle rising arpeggio.
- Light "whoosh" on each transition, no voice-over.

Export: MP4, 1080p, 30fps, loop-ready.
```

## PROMPT 4 : PRÉSENTATION INVESTISSEURS / PARTENAIRES

```
Rôle : Tu es le Directeur Stratégique d'Opays Tech.

Crée une présentation de [X] slides pour [CIBLE : investisseurs / partenaires / 
institutions]. Le contenu doit s'appuyer exclusivement sur le Livre Blanc 
officiel.

Structure recommandée :
1. Le Problème (friction opérationnelle et dépendance technologique).
2. La Solution Opays (Ingénierie de l'Efficience par l'IA).
3. Le Modèle (Studio pour le ROI + Labs pour l'intérêt général).
4. La Preuve (Opays HQ comme démonstration vivante).
5. L'Équipe (gouvernance Commando, vesting).
6. L'Appel (ce que le partenaire gagne en rejoignant Opays).

Ton : Ambitieux mais ancré dans le réel. Pas de promesses non chiffrées.
```
